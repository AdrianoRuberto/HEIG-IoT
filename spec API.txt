Spec APIs

2 APIs:
	- /api/stat (endpoint out)
	- /api/event (endpoint in)

API out:
	Renvoie le dataset pour les statistique/graphs
	Paramètres:
		- granularity [enum (year / month / day / hour)]
        - from [int] (timestamp)
        - to [int] (timestamp)

API in:
    Accepte des événements contenant les infos essentielles aux statistiques
    Paramètres:
        - timestamp [int]
        - type [enum (in / out)]
        - id [string] (hash)


Sample:
    API in:
        - timestamp=1234567890
        - type=in
        - id=acksbvakzsbvuasbvivrsvnavn

    API out:
        - granularity=day
        - from=1234567890
        - to=2234567890
